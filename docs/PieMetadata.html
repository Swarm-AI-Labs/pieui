PieMetadata

Канонический dump-формат компонента карточки — параллельный, симметричный, машино-разбираемый для pieui и pie.

Python · pie card dump-metadata TypeScript · pieui card dump-metadata

Поля

name files packages relativeImports events propsSchema propsCode eventsPropsSchema eventsPropsCode inputPropsCode inputPropsSchema ajaxList

name

string

Имя компонента. Должно быть идентично в обоих экосистемах.

Python

Имя класса в <component_name>.py, нормализованное через normalize_class_name. 

"SimpleCard"
TypeScript

Имя директории piecomponents/<Name>/ и аргумент registerPieComponent({ name }). 

"SimpleCard"

files

string[]

Абсолютные пути локальных файлов компонента.

Python 
[
  "/abs/path/components/simple_card.py"
]
TypeScript 
[
  "/abs/path/piecomponents/SimpleCard/index.tsx",
  "/abs/path/piecomponents/SimpleCard/types.ts"
]

packages

string[]

Пакеты-зависимости. Строки должны быть валидными аргументами для bun add / uv add.

Python

Из import x / from x import … в файлах компонента, отфильтровано от stdlib и проектных модулей. Версия из pyproject.toml или importlib.metadata. 

["pydantic>=2.5", "httpx"]
TypeScript

Из import … from "<pkg>", отфильтровано от relative-импортов. Версия из package.json. 

["react@^19.0.0", "clsx@^2.1.0"]
⚠ Версионная часть опциональна, но рекомендуется. При апдейте на чужом проекте — корректно установится тот же major.

relativeImports

string[]

Абсолютные пути проектных файлов, импортируемых компонентом, но не входящих в его собственные файлы.

Python

Резолв относительных импортов (from ..utils import foo) и абсолютных проектных (from myapp.shared import bar) к абсолютным путям файлов. 

["/abs/path/utils/helpers.py"]
TypeScript

Резолв import … from "./foo", "../bar", "@/x" через tsconfig.paths. 

["/abs/path/src/utils/format.ts"]

events

string[]

Имена реально отправляемых событий.

Python

Из get_supported_events() или из имен create_<event>_event. Берём то, что реально шлётся — пересечение источников. 

["click", "submit"]
TypeScript

Ключи объекта methods = { … } внутри <PieCard methods={…} />. 

["click", "submit"]

propsSchema

JSONSchema

JSON Schema интерфейса данных компонента. Все ключи — camelCase.

Python

Генерируется из @dataclass class SimpleCard. snake_case → camelCase при сериализации.

TypeScript

Генерируется из interface SimpleCardData через typescript-json-schema. 

{
  "type": "object",
  "properties": {
    "title": { "type": "string" },
    "buttonLabel": { "type": "string" }
  },
  "required": ["title"]
}

propsCode

string

Исходный код типа props.

Python

Весь файл с dataclass карточки целиком.

from dataclasses import dataclass

@dataclass
class SimpleCard:
    title: str
    button_label: str = "Click me"
TypeScript

Только блок interface SimpleCardData { … }. 

interface SimpleCardData {
  title: string;
  buttonLabel?: string;
}
JSON-сериализуемые типы (string/number/bool/null/object/array) + Any | unknown и Date | datetime. Никаких Callable, asyncio.Task, FunctionExpression, etc.

eventsPropsSchema

Record<string, JSONSchema>

Словарь eventName → JSONSchema описывающий ожидаемый payload каждого события. 

{
  "click": {
    "type": "object",
    "properties": { "x": { "type": "number" } },
    "required": ["x"]
  }
}

eventsPropsCode

Record<string, string>

Словарь eventName → исходник.

Python

Весь исходник функции def create_<event>_event(self, …):. 

def create_click_event(self, x: int, y: int) -> ClickEvent:
    return ClickEvent(x=x, y=y)
TypeScript

Только блок интерфейса для payload-а события.

interface ClickEventInput {
  x: number;
  y: number;
}

inputPropsCode

string | null

Код типа/dataclass для stored={} словаря, передаваемого в <PieCard>. Обычно у InputSimpleCard входной тип называется InputSimple.

Python 
@dataclass
class InputSimple:
    value: str = ""
TypeScript 
interface InputSimple {
  value: string;
}
Возвращает null, если у компонента нет input-типа.

inputPropsSchema

JSONSchema | null

JSON Schema того же input-типа. null если нет. 

{
  "type": "object",
  "properties": { "value": { "type": "string" } }
}

ajaxList

string[]

Имена ajax-pathname полей, объявленных в компоненте.

Python

Поля dataclass с префиксом pathname_. 

["pathname_click", "pathname_delete"]
TypeScript

Поля интерфейса с префиксом pathname (camelCase). 

["pathnameClick", "pathnameDelete"]
camelCase vs snake_case — это разные ключи, но смысл один. Конвенция нейминга остается локальной для каждой экосистемы.
